# Diagram Editor

ウェブベースのダイアグラムエディタです。

東京公共交通オープンデータチャレンジのAPIを使ってデータを読み込むことができます。

## デモアプリ

以下のページでデモアプリを試用できます。

[https://yamagame.github.io/diagram-editor](https://yamagame.github.io/diagram-editor)

## 準備

npm コマンドを使って node_modules を作成します。

```
$ npm install
```

## ビルト

以下のコマンドでビルドします。ビルドすると docs フォルダにアプリが作成されます。

```
$ npm run build
```
